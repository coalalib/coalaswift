    Declaration

    Swift

    func log(_ message: String, level: LogLevel, asynchronous: Bool)

    Parameters

    message

    The message to be logged.
    level

    Level of log message importance.
    asynchronous

    Describes how time-critical log message is. If set to true, logging can be delayed until later. If set to false, message should be logged immediately.
